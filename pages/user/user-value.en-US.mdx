# User Value

## Definition

-  What is value: Value is solving problems.
-  What is user value: User value, in short, refers to the value generated when users solve problems in scenarios through the use of a product.
-  Common misconception: Product features are not equal to user value.

## Core Values of Content and Community Products

-  First Layer: Emotional Value. Beautiful scenery, attractive people, and nice photos bring pleasure.
-  Second Layer: Practical Value. Efficiently knowing useful information, buying cheap items.
-  Third Layer: Social Value. Making a few friends, sparking some resonance, and surprising delights.
